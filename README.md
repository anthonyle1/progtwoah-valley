Progtwo-ah Valley is a Stardew Valley-inspired farming simulator built in C++17 and SFML 3, used to showcase SFML features to prepare UF COP3503C students for their final Minesweeper Project.

Assets
---
All assets used in Progtwo-ah Valley should be provided in the [cmake-build-debug/assets](https://github.com/anthonyle1/progtwoah-valley/tree/master/cmake-build-debug/assets) folder. 

### Fonts

Fonts are pulled from [Google Fonts](https://fonts.google.com/). 
- Momo Trust Display: https://fonts.google.com/specimen/Momo+Trust+Display
- Dot Gothic 16: https://fonts.google.com/specimen/DotGothic16
